## Technical Highlights

* **Microservices Architecture**: Architected and containerized the Grading Management System (GMS), a 6-service platform deploying FastAPI, React, MySQL, RabbitMQ, and Kubernetes configurations. See [GMS Project Page](https://yuvraj-rathod-1202.github.io/yuvraj.dev/projects/gms.html).
* **Graph Machine Learning**: Engineered DecompGTI, a graph tool-routing framework fine-tuned on Qwen2.5-7B using LoRA SFT and deployed as a local FastMCP server, achieving 100% tool routing accuracy. See [DecompGTI Project Page](https://yuvraj-rathod-1202.github.io/yuvraj.dev/projects/decompgti.html).
* **Database Systems**: Developed ChiralDB, an autonomous PostgreSQL SQL/JSONB database engine featuring statistical heuristics and lock-free datatype drift migrations. See [ChiralDB Project Page](https://yuvraj-rathod-1202.github.io/yuvraj.dev/projects/chiral_db.html).
* **Educational AI Tools**: Co-developed ElevateEd AI, a multimodal study kit generator utilizing Gemini models, securing the nationwide Runner-Up prize at the upGrad CodeEd AI Hackathon. See [ElevateEd AI Project Page](https://yuvraj-rathod-1202.github.io/yuvraj.dev/projects/elevateed.html).
* **Event Infrastructure**: Directed digital platforms as Tech Coordinator for TEDxIITGandhinagar, building secure Razorpay payment webhooks and a real-time web app for location QR-checkpoint scanning. See [TEDxIITGandhinagar Project Page](https://yuvraj-rathod-1202.github.io/yuvraj.dev/projects/tedxiitgandhinagar.html).
